# Redis Learning Project

This project is built to learn and practice Redis concepts in a simple Node.js environment.

## What this project is about

- Learn the basics of Redis
- Understand key-value storage
- Practice connecting Redis with Node.js
- Explore common Redis commands and use cases

## Topics covered

- Installing and running Redis
- Connecting to Redis from Node.js
- Storing and retrieving data
- Working with keys, values, and expiration
- Basic Redis operations such as SET, GET, DEL, and EXPIRE

## Getting started

1. Install Redis on your machine.
2. Start the Redis server.
3. Run the project files using Node.js.

## Example commands

```bash
redis-cli
SET name "Redis"
GET name
DEL name
```

## Purpose

This repository is intended for learning purposes only and helps build a strong foundation in Redis.
